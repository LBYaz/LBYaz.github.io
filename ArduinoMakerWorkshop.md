# Develop Arduino programs using VScode

(2026/04/06)

## 安装

直接在 VScode 的扩展菜单里搜索 Arduino Maker Workshop 安装即可

安装后打开Arduino Maker Workshop设置

先看一下AduinoIDE的首选项中项目文件夹地址，将其复制到下图路径
![](imageArduinoMakerWorkshop\image1.png)

## 使用

创建一个项目文件夹，用 VScode 打开文件夹，创建一个 .ino 扩展名的文件

写好测试代码

选择自己的开发板和端口
![](imageArduinoMakerWorkshop\image2.png)

Compile：编译
Compile（Clean）：清空已编译的目标文件，全部重新编译
Upload：上传到开发板
